# Introducción a React Hooks

Bienvenido a esta serie sobre React Hooks. En esta primera parte, vamos a explorar qué son los hooks y por qué son tan importantes.

## ¿Qué son los React Hooks?

Los hooks son funciones especiales que te permiten "enganchar" características de React en componentes funcionales.

```jsx
import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);
  
  return (
    
      Has hecho clic {count} veces
      <button onClick={() => setCount(count + 1)}>
        Incrementar
      
    
  );
}
```

## Hooks más comunes

| Hook | Propósito |
| -------- | -------- |
| useState | Manejar estado local |
| useEffect | Efectos secundarios |
| useContext | Consumir contexto |
